Dr. Coppelius's Secret Program
Difficulity : ★ ★ ☆ ☆ ☆

It seems that Dr. Coppelius needs our help. He has a software that can restore his own creation, but unfortunately, his computer is badly damaged so he only can access an .exe file and a picture. to recover all of the program, he needs to assemble all formula names he make to activate the restore point. For a note, Dr. Coppelius has a habit to hidden a formula name in any file, since his program is dangerous. After some discussion, the full software can be recovered if all parts are resembling together. Can you find the first formula hidden in these evidence ?

P.S : you don't need reverse engineering techniques in here. it is a simple forensic
