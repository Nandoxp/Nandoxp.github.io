Social Engineering


Social Engineering Life Cycle

Source :
https://www.imperva.com/learn/application-security/social-engineering-attack/

Social Engineering adalah aktivitas maligant dengan memanfaatkan interaksi manusia. SocEng memakai manipulasi psikologikal utk menipu user utk membuat human error, terus bisa leak sensitive information
Contoh attacknya :

  • Scareware
  • Baiting
  • Phishing
  • Quid Pro Quo


    • Network Forensic Toolkits

  • Firewall

    Program yg ditempatkan pada gateway network dari sebuah server dan bertugas melindungi resource private network dari network lain. Firewall biasanya melakukan filtering paket yg akan masuk ato mau keluar.

  • IDS

    Utk mendeteksi adanya intrusi ke network kita. Ada 2 jenis IDS, mau host based / network based. Terus cara alertnya mau by anomali ato by signature file?

    • Tools Network Forensic yang sering digunakan adalah :
  • Wireshark
  • NetIntercept
  • Netwitness Investigator
  • Snort


    • Jenis Attack pada Network :

  • DDOS
  • Identity Spoofing (by ARP / by IP)
  • MITM
  • Soceng
  • Insider Attack


    • 4 Step of Incident Response :

  • Preparation
  • Detection + Analysis
  • Containment, Eradication, Recovery
  • Post Incident Activity
