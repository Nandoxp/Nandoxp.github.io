    Storage & Memory

    Ada beberapa cara untuk menulis data ke media penyimpanan, yaitu:

  • Electromagnetism (kalo partikel kena magnet baca 1, ga kena yaudah baca 0)
  • Microscopic electrical transistor – flash (kalo transistor dicharge baca 1, ga dicharge baca 0)
  • Reflecting light – CD, DVD

    • Media penyimpanan data punya usianya masing-masing, entah utk menyimpan data jangka waktu pendek, bisa aja untuk nampung data temporary, bisa juga untuk keep data permanen (long term keeping). dan disini masih dibagi lagi ada 2 jenis memori, yaitu :

  • Volatile – data ada saat device masih dialiri listrik. Kalo mati datanya hilang
  • Non-volatile – data akan tetap, tersimpan walau media penyimpanan sudah dimatiin


    • File Extensions & file signatures :

    File Extension ga jadi jaminan filenya sesuai dgn extensionnya, kan bisa aja direname. Maka dari itu diperlukan namanya file signature, jadi kt bisa tau file apa yg sebenarnya skrg kita mau coba buka.

    File System

    File system bertugas utk merapikan sistem storage kita. Jd dia ini bs track free space di drive. File sistem dibagi atas 2 jenis, yaitu:

  • FAT – File Allocation Table (ga bs recover error diskrelated)
  • NTFS – New Technology File System (bs recover error disk-related)

    • Tiap Sector dari sebuah drive bisa nampung data sebanyak 512 bytes.


    Computer Environment

  • Standalone

    Device mampu bekerja tanpa bergantung pada resource hardware lain yang artinya devicenya ngga terintegrasi dengan device lainnya.

  • Networked

    Device-device saling terintegrasi 1 sama lain dalam 1 jaringan. server akan memproses request dari client.

  • Mainframe

    Device-Device terintegrasi dengan komputer mainframe - server room.
